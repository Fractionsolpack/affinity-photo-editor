# Affinity Photo Editor

Professional photo editing software offering Photoshop-level capabilities at a one-time price. Built for photographers, retouchers, and creative professionals.

## What Sets It Apart

> Affinity Photo delivers a complete photo editing workflow without ongoing costs. From RAW development to advanced compositing, it handles everything in a single application.

- Full RAW development with lens corrections and noise reduction
- HDR merge, focus stacking, and panorama stitching
- Frequency separation and liquify tools for retouching
- 32-bit HDR editing pipeline
- Batch processing for high-volume workflows
- PSD file compatibility with layer support
